# 个人知识库 (RAG)

你整天都在阅读文章、推文和观看视频，但永远找不到上周看到的那个东西。书签堆积如山，变得毫无用处。

此工作流从你保存的所有内容构建一个可搜索的知识库：

• 将任何 URL 拖入 Telegram 或 Slack，它会自动摄取内容（文章、推文、YouTube 转录、PDF）
• 对你保存的所有内容进行语义搜索：“关于智能体记忆我保存了什么？”返回带有来源的排名结果
• 供给其他工作流——例如，视频创意流水线在构建研究卡片时查询知识库以获取相关的保存内容

## 所需技能

- [knowledge-base](https://clawhub.ai) 技能（或使用嵌入构建自定义 RAG）
- `web_fetch` (内置)
- 用于摄取的 Telegram 主题或 Slack 频道

## 如何设置

1. 从 ClawdHub 安装 knowledge-base 技能。
2. 创建一个名为 "knowledge-base" 的 Telegram 主题（或使用 Slack 频道）。
3. 提示 OpenClaw：
```text
当我在 "knowledge-base" 主题中放置 URL 时：
1. 获取内容（文章、推文、YouTube 转录、PDF）
2. 将其摄取到知识库中并带有元数据（标题、URL、日期、类型）
3. 回复确认：摄取了什么以及块计数

当我在该主题中提问时：
1. 语义搜索知识库
2. 返回带有来源和相关摘录的最佳结果
3. 如果没有好的匹配，告诉我

另外：当其他工作流需要研究（例如，视频创意、会议准备）时，自动查询知识库以获取相关的保存内容。
```

4. 通过放置几个 URL 并询问类似“关于 LLM 记忆我有什么资料？”的问题来测试它。
