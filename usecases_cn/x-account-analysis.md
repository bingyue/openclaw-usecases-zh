# X 账号分析 (X Account Analysis)

有许多网站旨在为你提供 X 账号的定性分析。虽然 X 已经为你提供了一个**分析 (analytics)** 部分，但它更侧重于展示你的表现数据。

但是，定性分析侧重于帖子的质量，而不是性能统计数据。你可以从这种类型的分析中获得的一些见解：
- 让我的帖子病毒式传播的模式是什么？
- 我谈论什么话题最能获得互动？
- 为什么我有的帖子有 1000+ 点赞，但有时帖子只有 <5 个点赞？我做错了什么？

有许多网站和应用程序旨在为你提供 X 分析，但它们专注于统计数据。大概只有 1-2 个网站允许你与 AI 交谈以了解你的表现。

但现在你可以使用 OpenClaw 为你进行此分析，而无需在这些网站上支付 $10-$50 的订阅费。

## 你需要的技能
Bird Skill。`clawhub install bird`（它已预先捆绑）

## 如何设置
流程如下：
1. 确保 Bird skill 正在工作。
2. 为了安全和隔离，你最好为你的 ClawdBot 创建一个新帐户。
3. 使用你的 X 帐户进行身份验证。在 Chrome/Brave 中登录 x.com，并提供正确的 cookie 信息（`auth-token`, `ct0`），以便 OpenClaw 可以访问你的帐户。
4. 要求 OpenClaw 查看你的真实帐户，获取最近的 N 条推文，并向其询问你喜欢的任何问题。或者，你可以要求它为你编写特定的脚本。
