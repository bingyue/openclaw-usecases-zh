# 基于电话的个人助理

通过 OpenClaw 实现语音交互，使你可以通过电话呼叫你的 AI 代理。这对于在外出时无需盯着屏幕即可捕获想法、查询日历或管理任务非常完美。

## 它是如何工作的

1. **电话提供商 (Twilio / Telnyx)**：你拨打一个专用号码。
2. **语音网关 (Vapi / Bland AI)**：接听电话，将你的语音转录为文本，并将其发送给 OpenClaw。
3. **OpenClaw**：
   - 接收文本提示。
   - 检查日历、待办事项列表或知识库。
   - 生成响应。
4. **语音网关**：将 OpenClaw 的文本响应转换回语音并播放给你。

## 为什么使用它

- **完全免提**：非常适合开车或散步时使用。
- **更自然的交互**：与其打字，不如直接与你的数字大脑对话。
- **即时访问**：无需打开应用程序或解锁手机；只需拨打电话。

## 设置示例

### 1. 获取电话号码
从 Twilio 或 Telnyx 购买一个号码。

### 2. 配置语音网关 (例如 Vapi)
将你的电话号码指向 Vapi 的 SIP/语音端点。配置 Vapi 将转录内容作为 webhook 发送到你的 OpenClaw 实例。

### 3. OpenClaw 技能
确保你的 OpenClaw 具有处理对话输入的技能。

```yaml
name: phone-handler
description: 处理来自电话呼叫的语音输入
actions:
  - name: process_speech
    script: |
      #!/bin/bash
      USER_INPUT="$1"
      # 调用 LLM 确定意图
      # 执行日历查找、任务添加等
      # 返回文本响应
```

### 4. 提示
系统提示应针对语音进行优化（简洁、对话式）。

```text
你现在可以通过电话访问。当我来电时，问候我并询问我有什么可以帮忙的。
针对日历查询："What's on my calendar today?"（今天我的日历上有什么？）
针对 Jira 更新："Show me my open tickets"（显示我未关闭的工单）
针对网络搜索："Search for latest news on AI agents"（搜索关于 AI 代理的最新新闻）
```

## 扩展思路

- **每日简报电话**：安排 OpenClaw 每天早上给你打电话，朗读你的日程安排和重要新闻。
- **紧急升级**：如果服务器宕机，让 OpenClaw 给你打电话并大声朗读错误日志。
- **会议模拟**：在重要会议前与 OpenClaw 通话以练习你的演讲点。
