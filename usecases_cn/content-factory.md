# 多智能体内容工厂

你是一个内容创作者，在多个平台上兼顾研究、写作和设计。每一步——寻找热门话题、撰写脚本、生成缩略图——都会消耗你一天中的数小时。如果有一个专门的智能体团队通宵处理所有这些工作会怎样？

此工作流在 Discord 内部设置了一个多智能体内容工厂，不同的智能体在专用频道中处理研究、写作和视觉资产。

## 功能介绍

- **研究智能体** 每天早上扫描热门故事、竞争对手内容和社交媒体，寻找最佳内容机会
- **写作智能体** 选取最佳创意并撰写完整的脚本、推文串或新闻通讯草稿
- **缩略图智能体** 为内容生成 AI 缩略图或封面图像
- 每个智能体都在自己的 Discord 频道中工作，保持一切井井有条且可审查
- 按计划自动运行（例如，每天上午 8 点），这样你醒来就能看到完成的内容

## 痛点

内容创作有三个阶段——研究、写作和设计——大多数创作者都在手动完成这三个阶段。即使使用 AI 写作工具，你仍然需要一次提示一个。该系统将智能体链接成一个流水线，一个智能体的输出供给下一个，完全解放双手。

## 所需技能

- 与多个频道的 Discord 集成
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似工具用于社交媒体研究
- 本地图像生成（例如，Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于 RAG 驱动的研究）

## 如何设置

1. 设置一个 Discord 服务器（或让 OpenClaw 为你做——只需说“为我们设置一个 Discord”）。

2. 为每个智能体创建频道：
   - `#research` — 热门话题和内容机会
   - `#scripts` — 书面草稿和大纲
   - `#thumbnails` — 生成的图像和封面艺术

3. 提示 OpenClaw：
```text
我想让你在 Discord 内部为我建立一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体 (#research)：每天上午 8 点，研究热门故事、
   竞争对手内容以及在我的利基市场中社交媒体上表现良好的内容。
   发布前 5 个内容机会及来源。

2. 写作智能体 (#scripts)：从研究智能体那里获取最佳创意
   并撰写完整的脚本/推文串/新闻通讯草稿。发布在 #scripts 中。

3. 缩略图智能体 (#thumbnails)：为内容生成 AI 缩略图或封面图像。
   发布在 #thumbnails 中。

让它们的所有工作都在不同的频道中组织起来。
每天早上自动运行此流水线。
```

4. 根据你的平台进行定制：
```text
我专注于 X/Twitter 推文串，而不是 YouTube。更改写作智能体
以生成推文串而不是视频脚本。
```

## 关键见解

- 力量在于 **链接的智能体** — 研究供给写作，写作供给缩略图。你不需要单独提示每一步。
- Discord 频道使得分别审查每个智能体的工作并给出反馈变得容易，例如“脚本太长”或“更多关注 AI 新闻”。
- 你可以将其调整为任何内容格式：推文、新闻通讯、LinkedIn 帖子、播客大纲、博客文章。
- 运行本地模型进行图像生成（如在 Mac Studio 上运行 Nano Banana）可以降低成本并给你更多控制权。
